# Water Intake Tracker

Simple Streamlit app to log daily water intake (ml), show progress toward a 3L goal, and plot a 7-day hydration chart.

Run locally:

```bash
cd WaterTracker
python -m pip install -r requirements.txt
streamlit run app.py
```

Features:
- Log a water amount (ml) for a specific date.
- See today's total and a progress bar toward 3,000 ml.
- View a bar chart for the last 7 days and download CSV.
