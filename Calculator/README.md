# Simple Calculator

A beautiful, modern calculator web application built with Flask that performs basic mathematical operations.

## Features

- **Two Number Inputs**: Enter any two numbers
- **Operation Dropdown**: Select from Addition, Subtraction, Multiplication, Division, Modulus, and Power
- **Instant Calculation**: Results update automatically as you type
- **Real Calculator UI**: Modern, sleek design that looks like a real calculator
- **Error Handling**: Handles division by zero and invalid inputs gracefully

## Installation

1. Navigate to the Calculator directory:
```bash
cd Calculator
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Application

Run the Flask application:
```bash
python app.py
```

The calculator will be available at `http://localhost:5002`

## Usage

1. Enter the first number in the "First Number" field
2. Select an operation from the dropdown menu
3. Enter the second number in the "Second Number" field
4. The result will be displayed instantly in the calculator display
5. Click "Clear" to reset all inputs

## Supported Operations

- **Addition (+)**: Adds two numbers
- **Subtraction (-)**: Subtracts second number from first
- **Multiplication (ร)**: Multiplies two numbers
- **Division (รท)**: Divides first number by second
- **Modulus (%)**: Returns the remainder of division
- **Power (^)**: Raises first number to the power of second

## Technologies Used

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Modern CSS with gradients and animations
